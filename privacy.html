Política de Privacidad

Esta es una demo de sitio estático. Los formularios guardan datos localmente en tu navegador usando localStorage para simular la captura de leads. Si configuras un endpoint en assets/js/config.js, los datos se enviarán también a tu servicio externo (FormSubmit, Formspree, API propia, etc.).

No se comparten datos con terceros por defecto. Puedes borrar tus datos en Panel.
